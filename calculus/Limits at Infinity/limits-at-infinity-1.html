For \(x\geq 0 \), the horizontal line \(y=4\) is an asymptote for the graph of the function \(f\). Which of the following statements must be true?

Approach

A horizontal asymptote at \(y=4\) corresponds to a graph which approaches a value of \(y=4\) but never touches it. As the values of \(x\) increase without bound, the values of \(y\) approach 4.
