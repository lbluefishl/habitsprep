A retailer that sells footwear maintains a single database containing records with the following information about each item for sale in the retailer's store.

  • Item identification number
  • Footwear type (sneakers, boots, sandals, etc.)
  • Selling price (dollars)
  • Size
  • Color
  • Quantity available

Using only the database, which of the following can be determined?

Approach

Using the quantity available data, we can check if an item is still in the store. We do not know any of the buyer demographics and are unable to obtain information related to that. The database does not keep track of how many shoes are sold.
