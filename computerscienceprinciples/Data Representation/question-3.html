Consider the following numbers.

  • Binary 1100
  • Decimal 11
  • Hexadecimal D

Which of the following lists the numbers in order from least to greatest?

Approach

Hexadecimal is represented by 16 places: 0123456789ABCDEF. D is in the 13th place and therefore has a value of 13 in decimal.

Binary can be converted using base 2. 1100 represents:

$$ 1\cdot2^3 + 1\cdot2^2 + 0\cdot 2^1+0\cdot 2^0 = 8+4+0+0=12$$
