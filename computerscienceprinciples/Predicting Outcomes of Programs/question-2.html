Consider the following code segment, which uses the variables r, s, and t.

  • r ← 1
  • s ← 2
  • t ← 3
  • r ← s
  • s ← t
  • DISPLAY (r)
  • DISPLAY (s)

What is displayed as a result of running the code segment?

Approach

From the first three lines of code, we obtain the initial assignment of values. However, the values for r and s are changed in lines 4 and 5.
